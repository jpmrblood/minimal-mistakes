Install `yarn` first

```bash
$ sudo npm install -g yarn
```

Then, install `vue-cli`

```bash
$ sudo yarn global add @vue/cli
$ sudo yarn global add @vue/cli-init
```

After this, to make a new project:

```bash
$ vue init webpack my-new-site
```
